🔄 完整流程步骤

1
购物车选择商品

用户在购物车页面选择要购买的商品，点击"结算"按钮

2
跳转订单确认页面

系统保存选中商品到本地存储，跳转到订单确认页面

3
提交订单

用户在订单确认页面点击"提交订单"，系统调用API创建订单

4
支付处理

根据订单状态进行支付或跳转到订单列表页面

5
订单管理

用户可在订单列表页面查看订单状态，进行支付或其他操作

🔧 API 集成状态

当前配置: USE_MOCK = false API模式已启用
已集成的API接口：
  • orderApi.createOrder() - 创建新订单
  • orderApi.payOrder() - 订单支付
  • orderApi.getGoodsOrderList() - 获取订单列表
  • cartApi.getCartList() - 获取购物车数据
  • cartApi.updateCartItem() - 更新购物车项
